Android Animation

发表于 2015-05-17 | 分类于 Android |
个人觉得，用户良好的视觉体验不外乎两点：心旷神怡的UI风格与惊喜温馨的动画效果。这里，学习和总结下android动画。

#传统动画

##Tween 动画
主要有4种：渐变、平移、缩放、旋转

文件位置: res/anim/filename.xml

资源引用：
Java: R.anim.filename
XML: @[package:]anim/filename

###渐变< alpha>

  • android:fromAlpha 动画开始透明度
  • android:toAlpha 动画结束透明度

Float 值。0.0 表示完全透明，1.0 表示完全不透明，

###缩放< scale>

  • android:fromXScale
  • android:toXScale
  • android:fromYScale
  • android:toYScale

Float 值，动画起始到结束时，X、Y 坐标上的伸缩尺寸。0.0 表示收缩到没有1.0 表示正常无伸缩

  • android:pivotX
  • android:pivotY

Float值，缩放的中轴点 X/Y 坐标，可以实现动态调控件尺寸的效果。原点（0，0）表示左上角。如果想表示中轴点为图像的中心，属性值可定义成 0.5 或者 50%。

###平移< translate>

  • android:fromXDelta
  • android:toXDelta
  • android:fromYDelta
  • android:toYDelta

Float值（相对自身原始位置的像素值）或者百分比值（num%，num%p分别相对于自身或者父控件）。代表动画起始或者结束 X / Y 方向上的位置。例如，android:toYDelta=100% 表示在x方向上移动相对自己1倍的距离。

###旋转< rotate>

  • android:fromDegrees
  • android:toDegrees

Float值。代表代表起始和结束的角度（单位：度）

  • android:pivotX
  • android:pivotY

Float（相对于自身左边缘的像素值）值或者百分比（num%、num%p）。代表旋转中心的X、Y坐标值。其中X轴的相对方向是Left，Y轴的相对方向是Top。

  • android:fromDegrees
  • android:toDegrees
    Float值。代表开始和结束时的弧度位置(单位是度)

一般调用代码

  • android:fillAfter Boolean值。表示动画执行完后是否保持最后的状态
  • android:duration Int值，表示动画持续的时间，单位为毫秒。

###动画元素容器< set>
XML布局文件必须有一个独立的根元素，可以是 < alpha>,< scale>, < translate>, < rotate>, < set>（持有一组其它的动画元素，甚至可以是内嵌的 < set>元素）中的一个。

  • android:interpolator
    应用于动画的插值器。值为一个指定了插值器资源的引用。一般是平台缺省的插值器资源或者自定义的插值器资源。
  • android:shareInterpolator
    Boolean值，true：代表在所有的子元素中共享同一个插值器。false则相反。

##Frame 动画
文件目录：res/drawable/filename.xml
资源引用：
Java: R.drawable.filename
XML: @[package:]drawable.filename

###根元素
包含一个或者多个子元素

  • android:oneshot
    boolean值。true表示只执行一次动画，false表示循环执行

###子元素< item>

  • android:drawable
    Drawable 资源，用于这一帧的图片。
  • android:duration
    Integer 类型.该帧的时长，单位为毫秒 milliseconds。
