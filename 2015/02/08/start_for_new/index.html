伊始

发表于 2015-02-08 | 分类于 经历 |
不。这是个新的开始。刚刚开始。

  生活。总该做点自己喜欢的事情呢。

  初入大学，懵懵懂懂选择了软件工程专业，就注定了后来的一波三折。以前，轻微的二次元中二病。觉得，科技完全是可以改变世界的。事实上，确实如此。只是并不是人人都可以做到的。或者准确地说，只有极少数人能够做到。而我肯定不在极少数人里。

  此时，大学时光已过三年。一年，玩玩乐乐体验大学和城市生活，虽碌碌无为却一点都不后悔。一年，磕磕碰碰与代码相伴苦中作乐，虽学习缓慢却好像有好的趋势。一年，忙忙碌碌初入社会思索未来，虽有过迷茫却总算知道了大概的路。现在，大学还有一年。

  像大部分拥有赤子之心的程序猿一样，一直都想亲自去实践做一个有趣的玩意儿，或者一个网站，或者一个APP，或者一个平台，甚至一个算法。我喜欢简单，但不代表我没有做复杂事情的能力。因为，我知道我曾经认真做事时的模样。这一点，人人都可以做到。

  是时候该开始了。做自己喜欢的事情，顺便让自己变得优秀。就像我现在，席地而坐，晒着太阳，听着歌，喝喝茶，打打代码，偶尔发发呆，写写博客。生活虽然很现实，但最重要的是顺心而为，开心就好。最简单不过如此：不行就分，喜欢就买，多喝点水，重启试试。

  嘻嘻。仅以此，记录与分享技术与经历。
